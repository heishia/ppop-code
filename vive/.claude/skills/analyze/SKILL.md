# Analyze Skill

코드베이스를 분석하고 이해한다.

## 기능

- 프로젝트 구조 파악
- 의존성 분석
- 코드 패턴 식별
- 잠재적 문제점 발견
- 리팩토링 포인트 식별

## 출력

분석 결과를 구조화된 형태로 반환:
- summary: 전체 요약
- structure: 프로젝트 구조
- issues: 발견된 문제점
- suggestions: 개선 제안

## 사용 시나리오

- 새 프로젝트 이해가 필요할 때
- 리팩토링 전 현황 파악
- 코드 리뷰 시작 전
- 버그 원인 추적 시
