# Verify Skill

코드 변경 결과를 검증한다.

## 기능

- 변경사항 diff 확인
- 테스트 실행 (있는 경우)
- 빌드 확인
- 린트 검사
- 타입 체크

## 검증 항목

1. 파일 변경 확인
   - 예상된 파일이 수정되었는지
   - 의도치 않은 파일 변경은 없는지

2. 코드 품질
   - 린트 에러 없음
   - 타입 에러 없음

3. 기능 동작
   - 테스트 통과
   - 빌드 성공

## 출력

- status: success | warning | failure
- changes: 변경된 파일 목록
- test_results: 테스트 결과 (선택)
- issues: 발견된 문제점
